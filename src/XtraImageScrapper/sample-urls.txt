# === TIPS FOR CREATING YOUR OWN LIST ===
#
# 1. One URL per line
# 2. Use full URLs starting with http:// or https://
# 3. Make sure URLs point to actual image files
# 4. Supported formats: .jpg, .jpeg, .png, .gif, .bmp, .webp
# 5. Files larger than 10MB will be skipped (configurable)
# 6. Use comments like this to organize your URLs
#
